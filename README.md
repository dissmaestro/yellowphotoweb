# MAGIC photo — сайт фотостудии

Сайт-витрина для фотостудии **MAGIC photo** (Воронеж): обработка и печать
фотографий, фотомагниты, холсты, фотокниги. Построен на **Vue 3 + TypeScript +
PrimeVue + Vite**.

## Возможности

- **3 страницы** с маршрутизацией (Vue Router):
  - `/` — Главная: hero, услуги, портфолио, популярные цены, призыв к действию.
  - `/prices` — Прайс-лист (взят из официального прайса) + форма заказа.
  - `/about` — О студии, процесс работы, команда и контакты с картой.
- ✅ **Согласие на обработку персональных данных (ФЗ-152)** — обязательный
  чекбокс в форме заказа (заявку нельзя отправить без согласия) и полный текст
  согласия в модальном окне, доступном из формы и из футера.
- Элегантный фирменный дизайн в золото-бронзовой палитре логотипа, адаптивная
  вёрстка, кастомная тема PrimeVue.
- Контакты, карта, портфолио и блок «о нас» — чтобы сайт выглядел завершённым.

## Запуск

```bash
npm install
npm run dev      # http://localhost:5173
```

## Сборка

```bash
npm run build    # проверка типов + production-сборка в dist/
npm run preview
```

## Структура

```
src/
  assets/        логотип и глобальные стили
  components/    шапка, подвал, диалог согласия
  views/         HomeView, PricesView, AboutView
  data.ts        прайс-лист, услуги, контакты
  router/        маршруты
  main.ts        инициализация PrimeVue и темы
```

> Цены и логотип взяты из материалов студии. Телефон, адрес и ИНН в `src/data.ts`
> заданы как заглушки — замените на реальные перед публикацией.
