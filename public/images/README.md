# Изображения сайта

Сюда складывайте картинки для сайта. Файл, положенный в эту папку,
доступен по простому адресу `/images/<имя-файла>` — импортировать его в коде
не нужно.

Например, файл `public/images/hero-main.jpg` используется в коде как `/images/hero-main.jpg`.

## Какие файлы ожидает сайт

Замените заглушки, положив сюда файлы с такими именами (формат .jpg, можно и .png/.webp —
тогда поправьте расширение в коде):

| Файл                        | Где используется                         | Рекомендуемый размер |
| --------------------------- | ---------------------------------------- | -------------------- |
| `hero-main.jpg`             | Главная — большое фото в первом экране    | ~640×800             |
| `hero-small.jpg`            | Главная — маленькое фото в первом экране   | ~360×360             |
| `gallery-portrait.jpg`      | Главная — портфолио «Ретушь портрета»      | ~600×750             |
| `gallery-color.jpg`         | Главная — портфолио «Цветокоррекция»       | ~600×750             |
| `gallery-art.jpg`           | Главная — «Художественная обработка»       | ~600×750             |
| `gallery-quality.jpg`       | Главная — «Улучшение качества»             | ~600×750             |
| `gallery-restore.jpg`       | Главная — «Реставрация фото»               | ~600×750             |
| `gallery-before-after.jpg`  | Главная — «До и после»                     | ~600×750             |
| `about-process.jpg`         | О студии — фото процесса обработки          | ~720×560             |
| `master.jpg`                | О студии — фото мастера                     | ~520×600             |

Пока файл не положен в папку, на его месте картинка просто не отобразится.
